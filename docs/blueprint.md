# **App Name**: Нэргүй Хуудас

## Core Features:

- Compliment Link Generation: Generate a unique, shareable link associated with a random UUID.
- Anonymous Compliment Submission: Allow users to submit anonymous, text-based compliments via a generated link.
- Profanity Filtering: Use AI tool with a positive language filter and negative word auto-block for compliments.
- Compliment Inbox: Display collected compliments in a private inbox viewable only by the link owner.
- Anonymous Confession Posting: Enable users to post anonymous confessions to a public feed.
- Public Confession Feed: Display the latest confessions in a feed with emoji reactions.
- Profanity and Abuse Filter: Implement a profanity and abuse filter for confessions. Add a report button and auto-hide reported posts.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke calmness.
- Background color: Off-white (#F5F5F5) for a clean, minimal look.
- Accent color: Light coral (#F2B7A1) for interactive elements, drawing attention without overwhelming the calm aesthetic.
- Body and headline font: 'PT Sans' for a modern yet approachable feel. 
- Simple, minimalist icons for reactions and navigation.
- Mobile-first, single-column layout with large, readable text.
- Subtle animations on reactions and transitions to enhance the emotion-first design.
